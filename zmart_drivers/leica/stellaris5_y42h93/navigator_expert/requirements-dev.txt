# CI install list for the Navigator Expert offline suite -- NOT an environment.
#
# Humans build ONE environment: the repo-root conda env (environment.yml via
# build_env.py), with the repo-root requirements*.txt as the documented pip
# fallback. This file exists only so GitHub CI (.github/workflows/
# navigator-expert.yml and target-acquisition.yml) can install the offline
# suite's traced dependencies lean and headless:
#     python -m pip install -r requirements-dev.txt
#     python run_ci.py
#
# Every heavy dependency below is a *traced* transitive import of the offline
# suite, not a guess -- the comment says which code reaches it. NOTE: pythonnet /
# the LAS X CAM runtime are deliberately absent; they are only needed for the
# live @hardware tests on a microscope PC. The offline suite runs anywhere
# (Linux/macOS CI included).
